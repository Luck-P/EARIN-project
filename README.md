# EARIN Final Project Github Repository

* **Topic:** Project 15 - *Heart Attack Prediction*
* **By:** Luck Poyet

## [Dataset Source](https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data "Kaggle Website")

## To Run `project_t15.py`

1. Install the dependencies from `requirements` in your environment.

2. Edit the `DATASET_PATH` placeholder with the location of the *UCI Heart Disease* combined dataset.

3. Run `bootcamp(x)`
* 0: without the `origin` feature
* 1: with the `origin` feature
